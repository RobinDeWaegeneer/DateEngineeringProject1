Begin met een nieuwe connectie te maken in mysql workbench als localhost met user root en het wachtwoord zou je moeten weten.
Daarna "OLTP-Database-script.sql" uit voeren zodat de database lokaal op u pc staat.

als je de tabellen kan zien en de naam van de database in het vet staat kan je "LoadAllFiles.sql" uit voeren.
!!!!! Hiervoor moet je wel zorgen dat de file All_YYYY_MM_DD correct is veranderd en in de map C:/ProgramData/MySQL/MySQL Server 8.0/Uploads/ staat.

bij verdere problemen stuur dm

Voor de data warehouse procedures te kunnen maken moet je eerst "DWH-Database-script.sql" uitvoeren, als dat gedaan is moet je zeker zijn
dat je die database aan het gebruiken staat (kan je zien als de naam van de database int vet staat)

als dat zo is kan je ergens bij die database "Store Procedures" vinden, dan kan je rechter muisklik op doen om 1 aan te maken en dan kopieer
je eerst de code van "FillDimDate" en vul je die daar in en dan doe je hetzelfde voor "FillDimFlight".

Bij problemen mag je opniew mij een dm sturen op discord.
